SDL 1945

Top-down scroller game made to test the creation of a personal 2D Engine with C(programming language),
SDL2(graphic library) and CMake(compiler).

It's a remake of the old videogame "1945: The Final Front"(upper image).

Although the game is not complete and it has some soft bugs, it works properly :)

Tools: C - SDL2 - CMake

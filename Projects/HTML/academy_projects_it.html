Academy Projects

Prototipi di videogiochi realizzati durante i miei anni accademici creando un piccolo motore grafico(camera system, game manager, scene manager, physics system, ...) attraverso il supporto di 2 librerie:

- Memory -> Simulazione del Memory in cui è possibile scegliere il numero di carte selezionando la quantità di righe e colonne;

- Square Invaders -> Demo basata sullo sparatutto arcade "Space Invaders";

- Lander -> Gioco in cui bisogna far atterrare la navetta senza distruggerla;

- Space Shooter -> Un semplice gioco sullo stile arcade-space shooter;

- Smoker -> Gioco di sopravvivenza in cui si combattono orde infinite di nemici;

- Boids -> Simulazione del comportamento degli stormi di uccelli durante il volo.

Strumenti: C# - Aiv_Draw - Aiv_Fast2D

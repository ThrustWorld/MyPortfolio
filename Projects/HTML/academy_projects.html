Academy Projects

Videogames prototypes done during my academic years creating a little engine(camera system, game manager, scene manager, physics system, ...) using 2 raw libraries:

- Memory -> Memory game simulation where you can choose the number of cards selecting the amount of rows and columns;

- Square Invaders -> Demo based on the arcade shooter "Space Invaders";

- Lander -> Game where you have to land your shuttle without destroying it;

- Space Shooter -> A simple arcade space shooter game;

- Smoker -> Survival game where you fight infinite enemies hordes;

- Boids -> Simulation subject to birds storm behaviour during the flight

Tools: C# - Aiv_Draw - Aiv_Fast2D

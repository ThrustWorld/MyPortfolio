Rumbling 20's

It's an endless-runner game projected during the Global Game Jam 2020.

What's the Global Game Jam? The GJJ is a hackathon where you have to create a game in less than 3 days with a main topic given when the game jam starts.
The team was composed by 6 members: 3 software developers, 2 game designers, 1 2D/3D artist.

The game has been thought to be played on Ipad, so the screen is vertical.

Furthermore, it has been my first game jam and my first game using Unity. Due to this, I created an updated and completed version of the game: Rumbling 20's Remake.

Tools: C# - Unity

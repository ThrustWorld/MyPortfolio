Rumbling 20's Remake

Tools: Unity - C#.

Platforms: Windows - Mac - Linux.

Genres/Topics: Endless runner.

About: In the updated version, I decided to refactor the entire old project to improve the code readability and the software performance. Mainly, I worked on:

  • A complete menu section
    (Play, Settings, Credits, Quit, ...).

  • A score system influenced by the obstacles.

  • A valuable audio system using different channels.

  • A saving system using .json files.

  • A pooling system to recycle obstacles and chunks of the map to not overload the memory.

Nightmare Before Eco Congress

Tools: Unity - C#.

Platforms: Windows - Mac - Linux.

Genres/Topics: Educational game - Exploration. Social and environmental issues through lobbying mechanisms.

About: It's an educational game where you will embody the role of a village mayor during a congress. Your mission is the decision to install or not wind turbines; your actions will have consequences. I dealt with:

  • The systems and the managers of the game
    (Audio, Options, Save, ...).

  • UI for the HUD, popups, menus and credits.

  • Gameplay programming for interactions, minigames and collisions.
